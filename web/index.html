Outils système
ffmpegDécodeur audio, requis pour l'analyse approfondie d'intégrité.

macOS : dans le Terminal, brew install ffmpeg (si « brew » est inconnu, installe Homebrew depuis brew.sh).

Windows : winget install ffmpeg en PowerShell, ou télécharge sur ffmpeg.org et ajoute le dossier bin au PATH.

Linux : sudo apt install ffmpeg (Debian/Ubuntu) ou l'équivalent de ta distribution.
—
fpcalcEmpreinte acoustique (Chromaprint), requis pour la détection de doublons par le son et la vérification AcoustID.

macOS : brew install chromaprint (le binaire fpcalc est inclus).

Windows : télécharge Chromaprint sur acoustid.org/chromaprint, décompresse, et ajoute le dossier contenant fpcalc.exe au PATH.

Linux : sudo apt install libchromaprint-tools (Debian/Ubuntu) ou l'équivalent de ta distribution.

Repère les fichiers audio cassés, tronqués ou illisibles
Rapide : vérifie l'en-tête et la structure du fichier. Instantané, sans décodage.

Approfondi : décode entièrement l'audio via ffmpeg pour repérer les fichiers qui s'ouvrent mais sont corrompus ou tronqués en profondeur. Bien plus lent — résultats mis en cache ensuite.
Nombre de fichiers décodés simultanément. Plus élevé = plus rapide, mais plus gourmand en CPU. 4 est un bon compromis ; monte à 8 sur une machine puissante.
